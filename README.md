# 🎬 Movie Recommendation System

## 📌 Project Overview

This project is a **Movie Recommendation System** that suggests similar movies based on user input. It uses machine learning techniques to find movies with similar features.

---

## 📂 Main Steps

### 1. Data Collection

Movie dataset is used which contains title, genre, cast, crew, and keywords.

### 2. Data Preprocessing

Unnecessary data is removed, missing values are handled, and important columns are combined.

### 3. Feature Engineering

Text data is converted into numerical form using **vectorization (CountVectorizer/TF-IDF)**.

### 4. Similarity Calculation

**Cosine similarity** is used to calculate how similar movies are to each other.

### 5. Recommendation System

User enters a movie name → system finds similar movies → returns top recommendations.
